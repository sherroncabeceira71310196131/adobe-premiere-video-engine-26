## 🧠 Overview

**PremiereVideo Engine** is a heavily optimized, asynchronous C++ toolkit specifically designed to inspect Premiere Pro project structures, extract referenced media assets, and parse timeline metadata without requiring the editor application to be running.

Whether you are organizing post-production archives, migrating editing projects, or backing up linked footage and sequence structures, PremiereVideo Engine bridges the gap between Premiere project data and your local filesystem quickly and natively.

## 🚀 Features

*   **⚡ Hyper-Fast Media Scanning:** Built for rapid parsing of project references and linked source assets across large editing directories.
*   **🔓 Standalone Architecture:** Does not require Adobe Premiere Pro to be open in order to inspect supported project data.
*   **📦 Deep Timeline Extraction:** Extracts sequence metadata, referenced clips, media paths, and project-side asset manifests directly.
*   **🔗 Automatic Dependency Resolving:** Maps linked footage, audio tracks, and asset relationships across project structures.
*   **🎨 Cross-Ecosystem Support:** Output integrates with archive workflows, render prep scripts, and custom media management tooling.
